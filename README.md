# Cybersecurity Portfolio

My name is James Collins, and this repository contains hands-on cybersecurity labs completed as part of my cybersecurity coursework. These labs demonstrate practical experience with vulnerability assessment, packet analysis, access control, encryption, attack vectors, security policy, monitoring, backup and recovery, and incident response.

## Featured Skills

- Vulnerability assessment with Zenmap/Nmap and Nessus
- Packet capture and traffic analysis with Wireshark
- Active Directory users, groups, and access control testing
- Encryption, key exchange, signing, and decryption
- Web attack concepts including XSS and injection
- Security policy implementation and baseline configuration
- Logging and monitoring with Windows Event Viewer, Snort, rsyslog, and Tripwire
- Backup, recovery, load balancing, and failover testing
- Incident response, PCAP review, disk image analysis, and forensic reporting

## Purpose

The purpose of this portfolio is to show hands-on cybersecurity work in a clear and professional way. Each lab includes screenshots and documentation of the tools, configurations, and security concepts practiced.

## Career Focus

I am focused on defensive cybersecurity, especially SOC analysis, incident response, network monitoring, and digital forensics. These labs reflect my interest in understanding how attacks happen, how systems are secured, and how evidence is analyzed during investigations.
